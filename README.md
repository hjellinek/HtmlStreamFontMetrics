# HtmlStreamFontMetrics

Metrics for the fonts used by [`HTMLSTREAM`](https://github.com/hjellinek/HtmlStream).

Font height, max ascent, and max descent are equal to those of the glyphs in [`XCCS`](https://en.wikipedia.org/wiki/Xerox_Character_Code_Standard)
3-0-0 charset 0.

As of file format 3, we are using the MCCS (XCCS++) character code standard.

The current batch of files was generated by and corresponds to [`WebFontMetrics`](https://github.com/hjellinek/WebFontMetrics) revision  `98a6ebb0df7a58267cf698d1a73f36d2cda4c5f4`.
